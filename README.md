# Currency App

## Tavsif
Currency App â€” foydalanuvchilarga valyuta kurslarini kuzatish va boshqarishda yordam beradigan keng qamrovli va qulay ilova. Real vaqt yangilanishlari va zamonaviy interfeys bilan foydalanuvchilar valyutalar o'rtasida osongina konvertatsiya qilishlari va tarixiy ma'lumotlarni ko'rishlari mumkin.

## Xususiyatlar
- **Real Vaqt Valyuta Kurslari**: Yangilangan valyuta kurslarini real vaqt rejimida oling.
- **Valyuta Konvertatsiyasi**: Turli valyutalar o'rtasida osongina konvertatsiya qiling.
- **Tarixiy Ma'lumotlar**: Tarixiy valyuta kurslari ma'lumotlarini tahlil qilish uchun ko'ring.
